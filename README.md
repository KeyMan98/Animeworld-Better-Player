# AnimeWorld Better Player
Il player migliore di sempre.

---

## Funzioni

| Funzione | Descrizione | Hotkey |
|---|---|---|
| Autoresume | L'episodio riprende in automatico dall'ultimo punto in cui è stato interrotto. | — |
| Restart | L'episodio riparte dall'inizio, per annullare l'autoresume. | `R` |
| Skip OP/EP | Salto in avanti di 1:25 minuti per OP/EP standard. | `O` |
| Annulla Skip | Salto indietro di 1:25 minuti, in caso di click accidentale. | `B` |
| Episodio Precedente | Pulsante per passare all'episodio precedente direttamente dal player. | `P` |
| Episodio Successivo | Pulsante per passare all'episodio successivo direttamente dal player. | `N` |
| Autoplay | Quando viene premuto Precedente o Successivo l'episodio parte in automatico. | `A` |
| Auto Fullscreen | Quando viene premuto Precedente o Successivo l'episodio va in automatico in fullscreen. | `S` |

---

## Requisiti

Richiede [Tampermonkey](https://www.tampermonkey.net/) o [Violentmonkey](https://violentmonkey.github.io/).
Assicurati che l'estensione abbia attiva la lettura dei siti e che siano consentiti gli script utente.
